Nuestros Materiales

proceso del molde

Se desarrollara un molde para los huevos lo cual dentro de este proceso de perfora la caja para automatizarlo con un motor de manera de realise mivimiento mas o menos como la de entrar y salir de una puerta.

Vendilador

se ocupara para mantener vendilado con aire la caja de huevos. y se conectara junto con el termostrato para evitar que la temperatura se mantenga a 37 grados, este material se reciclara de una placa de pc

Focos Encandesentes

serviran para iluminar la caja y asi iluminar los huevos.
